# 欧洲工作与云服务生态多维度对比报告

一份针对欧洲主要 44 个国家的所得税、生活成本、法定假期、云服务生态、销售薪资、气候、社会安全、交通便利度、英语通用度及年轻社区活跃度等多维度深度对比与智能匹配交互式报告。

## 🌟 核心功能

1. **量化对比雷达图**：动态对比任意两个欧洲国家的 9 大维度评分。
2. **移居目的地智能匹配系统**：根据用户对税率、假期、薪资、安全、英语水平等 9 大维度的自定义权重，实时计算最契合的欧洲国家。
3. **全量对比表格**：支持 44 国全维度数据搜索与表头动态排序。
4. **分类维度探索（Tabs）**：快速查看所得税、云生态、物价安全、气候交通及语言融入细节。

## 🚀 本地运行

双击打开 `index.html` 即可运行（单文件无缝依赖）。
